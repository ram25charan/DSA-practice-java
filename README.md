# DSA Practice in Java

This repository contains my structured practice of Data Structures and Algorithms using Java.
Problems are organized by topic with clean code and brief explanations of the approach.

## Topics Covered
- Arrays
- Strings
- HashMap
- Linked List
- Stack & Queue
- Trees
- Binary Search
- Basic Dynamic Programming

## Structure
Each folder represents a topic.  
Each file contains:
- Problem statement (brief)
- Approach
- Time & Space Complexity
- Java implementation

- Strengthen problem-solving skills
- Prepare for technical interviews
- Focus on clarity and correctness over quantity
